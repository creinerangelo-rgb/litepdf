# LitePDF — Free Online PDF Tools

Fast, private, and free PDF tools that run entirely in your browser.

## Tools Included
- Merge PDF
- Split PDF
- Compress PDF
- PDF to JPG
- JPG to PDF
- Rotate PDF
- Password Protect PDF
- Word Counter

## Tech Stack
- Next.js 14
- pdf-lib (client-side PDF manipulation)
- pdfjs-dist (PDF rendering)

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → "Add New Project"
3. Import your GitHub repo
4. Click "Deploy" — no settings to change
5. Connect your custom domain in Vercel dashboard

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000
